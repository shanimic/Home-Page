# Project-Web – HTML & CSS Website

## 📌 תיאור הפרויקט
זוהי עבודת Web הכוללת דף אינטרנט שנבנה באמצעות HTML ו-CSS בלבד.  
הפרויקט מציג מבנה עמוד בית סטטי ועיצוב בסיסי באמצעות קובץ CSS, כחלק מתרגילי קורס פיתוח אתרי אינטרנט.

## 🗂️ קבצי הפרויקט

להלן הסבר קצר על הקבצים שתמצאו בריפוזיטורי:

- **homepage.html**  
  קובץ ה-HTML הראשי של האתר, בו נמצא המבנה של הדף בית והתוכן שמוצג למשתמש.

- **homepage.css**  
  קובץ CSS שמכיל את כל עיצוב האתר — צבעים, טיפוגרפיה, פריסות ועוד.

- **chart.jpeg**   
  מכילה את התמונה של הגרף המוצגת באתר.
